Basics of TypeScript

In the following you will learn the basics of TypeScript developement. These modules will guide you through various examples, so you can begin to create your own projects!

Types in TypeScript

Types by Inference

Unlike JavaScript the TypeScript language was built to generate the types for the user automatically. Therefore, allowing programmers to assign variables to a particular value without specifying the type.

For Example, in TypeScript assigning a value to a variable would look like this,

let hellowWorld = "Hello World";

TypeScript can achieve this by knowing how JavaScript works. This allows the language to infer what type the user is assigning to a variable.
